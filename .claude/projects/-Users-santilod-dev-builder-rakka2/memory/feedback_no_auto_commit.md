---
name: No hacer commits automáticos
description: No commitear sin confirmación explícita del usuario
type: feedback
---

No hacer commits automáticamente al terminar una tarea de implementación.

**Why:** El usuario prefiere controlar cuándo se commitea.

**How to apply:** Terminar la implementación y mostrar el resultado, pero no correr `git commit` a menos que el usuario lo pida explícitamente.
